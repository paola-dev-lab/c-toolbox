# C Toolbox

Repository dédié à la construction de projets techniques en C, dans une logique de montée en compétence structurée.

Ce travail s’inscrit dans le développement d’un profil hybride combinant expertise fonctionnelle SIRH et compréhension approfondie des systèmes techniques.

---

## 🎯 Objectif

Approfondir la compréhension :

- de la logique bas niveau
- de la gestion mémoire
- du parsing de données
- de la structuration d’outils en ligne de commande (CLI)
- de l’architecture modulaire

L’objectif est de renforcer la capacité à dialoguer efficacement avec les équipes techniques dans des contextes projets SIRH et systèmes.

---

## 🔧 Axes techniques travaillés

- Programmation en C
- Gestion mémoire (malloc, free, pointeurs)
- Manipulation de chaînes
- Lecture et traitement de fichiers
- Développement d’outils CLI
- Structuration et modularité du code
- Makefile et organisation de projet
- Validation mémoire (valgrind)

---

## 📦 Projets

### string_lab
Bibliothèque personnelle de manipulation de chaînes en C.  
Travail sur la gestion mémoire, la duplication, la concaténation et le parsing simple.

### text_analyzer
Outil en ligne de commande permettant d’analyser un fichier texte (statistiques de base).

### csvmini (en construction)
Mini outil de parsing CSV permettant la sélection de colonnes et le filtrage simple.

---

## 🧩 Approche

Chaque projet respecte les principes suivants :

- Structure claire des dossiers (src / include / tests)
- Makefile propre
- Gestion rigoureuse des erreurs
- Code lisible et modulaire
- Documentation synthétique
- Amélioration progressive

---

## 🚀 Vision

Développer une compréhension technique solide pour soutenir des projets SIRH complexes, en combinant :

- Analyse fonctionnelle
- Structuration des processus
- Compréhension des mécanismes système
- Dialogue fluide avec les équipes IT

Construire des systèmes robustes commence par comprendre leurs fondations.
